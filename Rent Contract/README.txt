Проект разрабатывался в рамках конкурсного задания чемпионата  Worldskills.

Реадизован следующий функционал:
1. Регистраци авторизаци пользователей 
2. Регистрация квартиры(жилая площадь)
3. Личный кабинет
4. Сделка аренды 
5. Просмотр всех доступных домов для аренды 
6. Изменение условий аренды
