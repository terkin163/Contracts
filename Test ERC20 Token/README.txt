Проект разрабатывался в рамках тестового задания за 8 часов.

Реализован следущий функционал:

1. Страндарт ERC20
2. Функция покупки токенов с разным курсом

Ревлизовано два варианта решения задания:

1. Покупка токенов сделана с возможностью покупки любого количева токенов. Но в данной реализации не корректно работает расчет стоимость
2. Покупка токенов сделана в виде лота за целое количесво эфира. Рабоатет более правильно но нет возможности купить 1 токен(только лот)
